# Fivestone

## 주제
오목프로그램

## 요구사항
1. 19*19 크기의 바둑판에 x좌표와 y좌표 값을 입력받아 돌을 두고 가로 혹은 세로 혹은 대각선으로 한플레이어가 5개를 완성하면 승리한다
2. 전적이 기록되고 출력된다
3. 돌의 모습을 바꿀 수 있다
